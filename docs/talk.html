How to Display an Image

for (x <- 0..9)
    for (y <- 0..9)
       plot (x, y, image[x][y])

Result

Flipped on the diagonal

Fix

for (y <- 0..9)
    for (x <- 0..9)
       plot (x, y, image[x][y])

Fix

for (y <- 9..0 step -1)
    for (x <- 0..9)
       plot (x, y, image[x][y])

Fix

for (y -> 9..0 step -1)
    for (x -> 9..0 step -1)
       plot (x, y, image[x][y])

There are 8 variations of this

I tried them all.

Multiple times.
